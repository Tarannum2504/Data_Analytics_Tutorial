# SQL Knowledge Base

A structured collection of SQL learning resources organized by skill level. This knowledge base covers everything from basic queries to advanced optimization techniques, with dedicated interview preparation materials.

## Structure

```
07-SQL/
├── 01-Basics/        → SQL fundamentals (SELECT, WHERE, aggregations, basic patterns)
├── 02-Intermediate/  → Joins, subqueries, CTEs, window functions, intermediate patterns
├── 03-Advanced/      → Query optimization, advanced SQL features, advanced patterns
└── Interview/        → Interview questions, common mistakes, SQL tricks
```

## How to Use

- Start with **01-Basics** if you're new to SQL or need a refresher.
- Move to **02-Intermediate** once you're comfortable with the fundamentals.
- Dive into **03-Advanced** for performance tuning and complex SQL techniques.
- Use the **Interview** section to prepare for SQL interviews.
